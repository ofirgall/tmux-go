# tmux-go
Go to your tmux session as quickly as possible

## Install
* Call `./src/tmux_go_attach.sh` at your end of your `.shellrc`
* Auto set your terminal title to `tmux-go-session:{session_name}`
* Install tmux-go plugin to your launcher
